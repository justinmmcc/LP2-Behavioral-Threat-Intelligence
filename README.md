# Project LostPiglet 2: Behavioral Threat Intelligence 

## Overview

Project LostPiglet 2 is a behavioral intelligence honeypot designed to analyze the interaction patterns between automated botnets and potential human threat actors.

This project explores how **cognitive friction**—introduced via psychological lures such as password-protected archives—affects attacker behavior and decision latency.

## Key Features

* Multi-phase honeypot deployment using Cowrie
* Behavioral filtering via deception artifacts ("Jim Harris" persona)
* Longitudinal data collection over 30 days
* Analysis of 25,000+ interaction events
* Distinction between automated bot activity and human-driven engagement

## Technical Stack

* Cowrie SSH Honeypot
* Linux-based cloud deployment
* Log aggregation & JSON telemetry
* Behavioral analysis pipeline

## Results

* Captured over 25,000 attack sessions
* Identified dominant botnet signatures (Echo Swarm, SSH brute force)
* Validated hypothesis: **Cognitive friction filters human actors from automated systems**
